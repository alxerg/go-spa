Welcome

Go SPA is a sample single page application (SPA) written in Go (golang) on the backend side, and Javascript with AngularJS on the frontend side.

The main goal is to provide the basic project structure for this kind of web application, even for the larger ones.

What's in it

Here is a quick list of what is covered by this sample application:

  • Token based authentication
  • Internationalization (i18n) and Localization (l10n)
  • Database communication
  • Database schema migration
  • Email sending
  • User registration and password reset logic

Backend

  • Integration with PostgreSQL database using gotk/pg
  • Easy http api using gotk/ctx
